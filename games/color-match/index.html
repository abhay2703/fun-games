🎯 Color Match

Score: 0 Best: 0 Time: 30s

📖 How to Play

A color name is shown in a colored font. Decide if the font color matches the word. Be fast — you have 30 seconds!

  • Press YES if the font color matches the word
  • Press NO if they don't match
  • Earn points for correct answers
  • Speed bonuses for quick responses

🎯 Skills & Benefits

Color Match exploits the Stroop effect to improve cognitive flexibility and processing speed.

  • Reflex: Quick decision making
  • Focus: Ignore conflicting information
  • Cognitive Flexibility: Process competing stimuli

Recommended for: Ages 7+
